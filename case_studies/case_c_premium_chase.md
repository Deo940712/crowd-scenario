# CASE C — 溢價 + 低殖利率的追高

> **這是情境排練，不是預測。** 用合成散戶原型排練「一檔熱門 ETF 溢價、殖利率又偏低，
> 但你很想追」的群眾反應。輸入是**手工整理的原型情境**，不是任何真實標的的資料；引擎
> 不預測漲跌，只演「若群眾長這樣、會怎麼看你的追高衝動」。重現：`python case_studies/report.py C`。

## 1. 情境背景

一檔熱門高股息 ETF 因為人氣，市價**高於淨值（溢價）**，同時**殖利率被墊高的價格壓到偏低**。
你（散戶）看它天天上漲、很想追。這是「FOMO 追高」的經典場景。我們問：「**如果**一群典型
散戶原型看到這個溢價+低息的組合，會潑你冷水還是陪你追？」

## 3. Personas 反應（default = aggregate_neutral）

stance 分布：**-1 五人 / 0 五人 / +1 零人**——**看空與觀望各半，無人看多**。

| stance | persona | 代表語音 |
| --- | --- | --- |
| -1 | 存股族 | 在相對高位存股意義不大，寧願等回檔就好一起。 |
| -1 | 殖利率派 | 擔心貼息與配息縮水，先減碼避風頭。 |
| -1 | 外資視角 | 調節部位、對沖風險，不追殺也不接刀。 |
| -1 | 媽媽存股社團 | 越漲越買、當作打折，但我當停扣觀望。 |
| -1 | 主力/中實戶 | 趁人氣高出貨、調節持股，先落袋。 |
| 0 | 當沖客 | 區間震盪不好做，先觀望減少進出。 |
| 0 | 槓桿 ETF 玩家 | 盤整最傷槓桿，暫時退場。 |
| 0 | 恐慌散戶 | 看不懂方向，先抱著不動。 |
| 0 | PTT/Dcard 風向 | 討論度冷清，沒什麼人在推。 |
| 0 | 定期定額新手 | 小額試單，還在學怎麼看。 |

**基本面派（存股、殖利率、外資、主力）明確偏空**（溢價高、息又低，不值得追）；純動能派
在缺乏動能訊號時**觀望**。沒有任何原型陪你追高。

## 7. 哪裡有用（What this told us）

- **對追高衝動的一記冷水**：你 positive、群眾 negative 的 HIGH divergence，明確點出
  「基本面派全在減碼、主力在出貨」——這正是 FOMO 時最該聽到的合成第二意見。
- **看空理由結構化**：存股族嫌貴、殖利率派怕貼息、主力趁人氣出貨——不是單一情緒，
  而是不同型態各自的理由。
- **和 CASE A 對照**：同一個 STOCK_TW pack，深折價高息（A）→ 群眾偏多，溢價低息（C）
  → 群眾偏空。方向由情境決定，不是預設立場。

## 8. 哪裡不能信（What this CANNOT tell us）

- **它不預測這檔會不會繼續漲。** 熱門股可以在「所有基本面派都看空」時繼續噴——群眾
  排練不是動能預測。
- **personas 是刻板原型**，不含真實資金流、消息面、主力真實意圖。
- **輸入是手工原型情境**；真實溢價/殖利率一變，結論可能不同。
- **『群眾潑冷水』不代表你追高一定錯**：這一層只幫你把「多數合成型態不陪你追」這件事
  攤開來看，決定權和後果都在你，不在引擎。
